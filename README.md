# Probopub

## About
*Probopub* is web app that allows end users learn more about United States case law, in a manner that is interactive and not convoluted.

## Technologies Used
- MERN
  - MongoDB
  - Express JS
  - React
  - Node JS
- Axios
- Mongoose
- CORS
- [Courtlistener API](https://www.courtlistener.com/api/rest/v3/)
